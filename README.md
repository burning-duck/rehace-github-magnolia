## Usage


### npm run

```
npm run start
```
- clean
- Watch all files in `_dev` for changes.
- Saves build bundles in `webreources`
- Start a livereload server on default port 35729


```
npm run build
```
- clean
- Builds a bundle from `_dev`.
- Saves build bundles in `webreources`


```
npm run clean
```
- Deletes `webresources`
